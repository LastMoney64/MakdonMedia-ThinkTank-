# 토론 세션

하나의 주제에 대해 여러 역할이 순서대로 독립적으로 의견을 제시한다.

## 진행 순서
1. **Analyst** — 관련 데이터/팩트 정리
2. **Devil's Advocate** — 반박, 리스크, 반대 시나리오
3. **Strategist** — 장기 관점에서의 판단
4. **종합** — 세 관점을 종합한 최종 판단

## 출력 형식
```
━━━ 토론: [주제] ━━━

📊 Analyst:
[데이터 기반 분석]

🔴 Devil's Advocate:
[반박과 리스크]

🎯 Strategist:
[장기 관점 판단]

━━━━━━━━━━━━━━━━━━

⚖️ 종합 판단:
[결론]

📋 추천 액션:
[구체적 다음 행동]
```

## 규칙
- 각 역할은 독립적으로 판단하라. 앞 역할 의견에 끌려가지 마라.
- 3명이 만장일치면 오히려 의심하라 — 놓친 관점이 있을 수 있음.
- 종합 판단에서 "어느 쪽이 더 무거운가"를 명확히 하라.
- 결론이 나오면 `decisions/`에 저장할지 물어라.
- 토론 과정에서 새 인사이트가 나오면 `domains/`에 저장 제안.
