# 텔레그램 자동화 에이전트 — 막돈방

텔레그램 슈퍼그룹(포럼 모드) 토픽별 콘텐츠를 자동 생성한다.

## 채널 구조
```
막돈방 (슈퍼그룹 + 포럼 모드)
├── 📌 공지
├── ☀️ 브리핑        ← 아침/저녁 브리핑
├── 🪙 코인분석      ← 코인 분석
├── 💭 KOL          ← KOL 인사이트
├── 🤖 AI           ← AI 리포트
├── 🌏 매크로        ← 매크로 리포트
└── 📊 일별요약      ← 일별 요약
```

## 품질 기준
- 모든 콘텐츠는 **팩트 기반** — 추측은 명시적으로 표기
- **두괄식** — 헤드라인이 핵심, 설명은 보충
- **실전적** — "그래서 어떻게?"에 답하는 내용 필수
- **일관된 톤** — 전문적이되 친근, 과장 없이 솔직
- 각 리포트의 **마지막 한마디**가 채널 정체성을 결정함

## 데이터 소스 우선순위
1. 공식 기관 (한은, Fed, BIS, IMF)
2. 주요 통신사/미디어 (Bloomberg, Reuters)
3. 온체인 데이터 (Glassnode, DefiLlama)
4. KOL/전문가 분석 (교차 검증 필수)

## GitHub Actions 연동
- `makdon-briefing` 레포의 워크플로우와 연결
- 각 콘텐츠 타입별 워크플로우 트리거
- 실패 시 재시도 (최대 2회) + 알림
