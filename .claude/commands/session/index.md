# 세션 관리

## 세션 시작 시
1. CONTEXT.md 읽기
2. memory/core.md 읽기
3. sessions/ 폴더에서 가장 최근 파일 읽기
4. 간단 브리핑 제공:
   - 현재 상황 요약
   - 지난 세션에서 이어갈 것
   - 오늘 체크할 것 (결정 재검토 시점 등)

## 세션 종료 시
1. 오늘 대화의 핵심을 `sessions/YYYY-MM-DD-주제.md`로 저장
2. 새 인사이트가 있으면 → `memory/core.md` 업데이트
3. 내 상황이 바뀌었으면 → `CONTEXT.md` 업데이트
4. 의사결정이 있었으면 → `decisions/`에 로그
5. 저장한 파일 목록을 알려줘라

## 세션 파일 형식
```markdown
# YYYY-MM-DD 세션: [주제]

## 논의한 것
- [핵심 내용 3-5줄]

## 결정/결론
- [있으면]

## 새로운 인사이트
- [있으면]

## 다음에 이어갈 것
- [미결 사항, 숙제]

## 업데이트된 파일
- [변경된 파일 목록]
```
