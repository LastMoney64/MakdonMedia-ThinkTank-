# 지식 인테이크 프로세스

외부 콘텐츠가 입력되면 아래 4단계 검증을 실행하라.

## 4단계 검증

### ① 출처 검증
- 1차 자료(원본 데이터, 공식 발표, 논문)인가?
- 2차 자료(누군가의 해석, 요약, 재가공)인가?
- 출처 신뢰도: 🟢공식기관/학술 🟡전문가/미디어 🔴불명/익명

### ② 근거 검증
각 주장에 태그를 붙여라:
- **[데이터]** 구체적 수치/통계가 있음
- **[의견]** 개인 해석, 전문가 견해
- **[추측]** 근거 없는 전망, 희망적 관측

### ③ 반대 시나리오
- 이 주장이 틀리려면 어떤 조건이 필요한가?
- 역사적으로 비슷한 주장이 틀린 사례가 있는가?
- 저자가 의도적으로 빠뜨린 관점이 있는가?

### ④ 적용성 (CONTEXT.md 참조)
- 내 현재 상황/포트폴리오/목표에 해당되는가?
- 당장 액션이 필요한가, 관찰만 할 것인가?
- 기존에 축적된 지식과 모순되는가, 보강되는가?

## 검증 후 처리
| 결과 | 액션 |
|------|------|
| 유의미한 인사이트 | `domains/해당분야/YYYY-MM-DD-키워드.md`에 저장 |
| 의사결정 필요 | `decisions/`에 초안 생성 |
| 아이디어 발견 | `ideas/`에 메모 |
| 가치 없음 | 저장하지 않고 이유 설명 |

## 저장 파일 형식
```markdown
# [제목/키워드]
- 날짜: YYYY-MM-DD
- 출처: [원본 출처]
- 신뢰도: 🟢/🟡/🔴

## 원본 요약
[3-5줄]

## 검증 결과
[4단계 검증 핵심 내용]

## 핵심 인사이트
- [내가 가져갈 것]

## 액션 아이템
- [ ] [해야 할 것]
```

## [2차요약] 태그 처리
유튜브 등 NotebookLM 경유 콘텐츠는 이미 AI가 한 번 해석한 것임.
→ 원본이 아니므로 주장의 정확성에 더 높은 검증 기준을 적용하라.
→ "원본 확인 필요" 태그를 붙여라.
